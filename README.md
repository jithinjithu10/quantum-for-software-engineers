# quantum-for-software-engineers
Learn quantum computing the way software engineers think

> Learn quantum computing using code-first explanations, intuitive models, and real-world security examples.

##  Why This Repository?
Most quantum resources assume physics or heavy math backgrounds.
This repository explains quantum computing **the way software engineers understand systems**.

##  What You Will Learn
- Qubits as stateful objects
- Quantum gates as pure functions
- Circuits as execution pipelines
- Measurement as probabilistic output
- Quantum Key Distribution (BB84)
- Why post-quantum cryptography matters
